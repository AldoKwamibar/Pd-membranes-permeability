The folder 'Dataset' contains three files:
- The 'Lattice parameters' Ms Excel worksheet describes how missing lattice parameters are calculated
- The 'oliynyk_extended_std_unit' (.csv) is the dataset used for CBFV fetaures calculations. It contains less columns than the original. See https://doi.org/10.1016/j.dib.2024.110178 for the original dataset.
- The Palladium_Alloy_Membranes dataset_QSPR_ML (.csv) is the dataset used for ML prediction
